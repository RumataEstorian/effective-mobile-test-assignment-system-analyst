# Исправленное ТЗ
1. Пользователь может добавить в корзину от 1 до 10 единиц* одного вида товара**. Отображение: название, картинка, количество единиц, цена за единицу и общая стоимость позиции (количество х цена за единицу).
2. При превышении лимита товаров одного вида, система показывает: "Вы превысили лимит товаров одного вида".
3. Пользователь может снизить количество товаров до 0, тогда товар удалится. Также рядом с товаром должна находиться кнопка для разового очищения позиции товара одного вида.
4. В корзине должно находиться не более 5 видов товаров. Отображение: товары должны быть расположены вертикальным списком.
5. При попытке добавить 6-ого вида товара система показывает сообщение: "Нельзя добавить более 5 видов товаров".
6. Суммарное количество всех товаров не должно превышать 20 единиц. В нижней части корзины должен быть расположен счётчик общего количества единиц товаров в корзине.
7. Правее счётчика общего количества единиц товаров в корзине должна присутствовать общая стоимость товаров.
8. При попытке добавить больше 20 единиц товаров система показывает: "Нельзя добавить более 20 товаров в одну корзину".
9. Цена товара в корзине фиксируется на N минут после изменения цены в каталоге. (Время N обсуждается с заказчиком)
10. Неоплаченная корзина автоматически очищается после N времени. (Время N обсуждается с заказчиком)
11. В корзине может проходить реклама других товаров в будни, в промежутках с чч.мм по чч.мм дневного времени и с чч.мм по чч.мм вечернего времени. Должен учитываться часовой пояс клиента. (Время чч.мм обсуждается с заказчиком).

Глосарий:
1. *единица товара - один физический экземпляр товара
2. **вид товара - уникальная позиция каталога
